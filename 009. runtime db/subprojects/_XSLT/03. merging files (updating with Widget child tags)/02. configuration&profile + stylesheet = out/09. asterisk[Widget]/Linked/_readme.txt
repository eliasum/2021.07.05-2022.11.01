При наличии во входном файле у любого тега ТОЛЬКО ОДНОГО дочернего тега Widget
в выходном файле будут атрибуты и дочерние теги тега Widget из файла профиля 
profile.xml (ПРИОРИТЕТ ДАННЫХ ИЗ ФАЙЛА ПРОФИЛЯ). 

При отсутствии в файле профиля и одновременном наличии во входном файле тега 
Widget в соответствующем месте, в выходном файле будут данные атрибутов и 
дочерних тегов тега Widget из входного файла.
